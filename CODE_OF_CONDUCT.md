# Code of Conduct

This project adheres to the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be kind and respectful.
- Give feedback on code, not on people.
- Assume good intent.
- Escalate concerns to the project maintainer via the email address in
  `action.yml` or via a private GitHub message.

The full text of the Contributor Covenant 2.1 is incorporated by reference
into this document. By participating in this project (filing issues, opening
PRs, commenting), you agree to abide by its terms.
